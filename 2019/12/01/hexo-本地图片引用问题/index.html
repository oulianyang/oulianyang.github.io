hexo_本地图片引用问题

发表于 2019-12-01

hexo4.0图片引用

步骤

  1. 在根目录下配置文件_config.yml 中 post_asset_folder:false改为true

  2. npm install https://github.com/7ym0n/hexo-asset-image –save

  3. 执行以上两步后在新建 hexo n xxxx博客名 就会同时在当前目录生成一个 同名文件夹，将需要的图片放入其中

  4. 本地测试需要重启本地服务,清空缓存
